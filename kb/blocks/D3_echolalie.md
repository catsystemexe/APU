
# Echolalie

Echolalie znamená, že dítě opakuje slova nebo věty, které slyšelo, místo aby vytvářelo vlastní odpověď.

Může jít o:
- okamžité opakování („Co chceš?“ → „Co chceš?“)
- odložené opakování (fráze z pohádek, her)

Nejde o „nesmyslné mluvení“, ale o způsob, jak dítě:
- zpracovává jazyk
- snaží se komunikovat
- udržuje kontakt

Dítě často:
- nerozumí plně významu
- používá naučené vzorce místo tvorby řeči

Klíčové:
- echolalie = „používám jazyk, jak umím“
- ne = „mluvím bez smyslu“
